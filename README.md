## Nima qiladi?

O'qituvchi dars o'tayotganda Zukko AI mikrofonni yoqib darsni tinglaydi va avtomatik ravishda:

- 🗺️ **Road Map** — dars mavzusidan tematik xarita tuzadi
- 📝 **Test savollari** — dars mazmunidan avtomatik quiz yaratadi
- ⚠️ **Xatolarni aniqlaydi** — ovozda noto'g'ri ma'lumot berilsa tizim o'zi topadi
- 📄 **PDF tahlili** — kitob sahifalarini yuklasangiz ham tahlil qiladi
- 📊 **Dashboard** — o'qituvchi va talabalar uchun alohida panel

## Texnologiyalar

- **Frontend:** React 19, TypeScript, Tailwind CSS, Vite
- **AI:** Google Gemini AI (audio + text tahlil)
- **Backend:** Firebase Auth + Firestore
- **Boshqa:** PDF.js, jsPDF, html2canvas, D3.js, Framer Motion

---

## Ishga tushirish

**Talab:** Node.js v18+
```bash
# 1. O'rnatish
npm install

# 2. .env.local faylini yarating
GEMINI_API_KEY=your_key_here
APP_URL=http://localhost:3000

# 3. Ishga tushiring
npm run dev
```

Brauzerda: `http://localhost:3000`
